## 3. 共通コンポーネント

| コンポーネント | 用途 |
|---|---|
| `SiteHeader` | 上部ヘッダー(ログイン状態で出し分け) |
| `BottomNav` | モバイル用5タブ: ホーム/検索/体験/診断/マイ |
| `TrustFooter` | 信頼性確保のフッター(SSL表記・匿名投稿対応・法務リンク) |
| `WardCard` | 病棟カード(人間関係★・残業・バッジ) |
| `ReviewCard` | レビューカード(年次・在籍状態・スコア4軸・バッジ) |
| `ReviewsWithFilter` | ★評価分布+フィルタ付きレビューリスト |
| `RatingDistribution` | ★5〜1の棒グラフ。クリックでフィルタ |
| `ProsCons` | レビュー集計からGood/Concernsを自動抽出 |
| `RadarChart` | SVGレーダーチャート(6軸) |
| `VerifiedBadge` | 現職/元職員Verifiedバッジ |
| `WardPhotoPlaceholder` | 雰囲気写真の4枚スライダー(現状はグラデプレースホルダ) |
| `ReviewEmptyCTA` | 30件未満警告+匿名投稿誘発 |
| `FavoriteButton` | お気に入り追加/解除(Server Action) |
| `HelpfulButton` | 参考になった(localStorage) |
| `ScoreBadge` | ★5段階表示 |
